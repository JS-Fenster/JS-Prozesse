## Design-Prinzipien

| Prinzip | Beschreibung | Beispiel |
|---------|--------------|----------|
| **Mobile-First** | Zuerst fuer Smartphone, dann Desktop | Monteur-App ist Hauptfokus |
| **Minimal Clicks** | Weniger Klicks = schneller | Budget-Angebot in 3 Klicks |
| **Contextual Actions** | Aktionen dort wo man sie braucht | "Anrufen" direkt beim Kunden |
| **Progressive Disclosure** | Nur zeigen was gerade relevant ist | Details erst bei Bedarf |
| **Konsistenz** | Gleiche Aktionen = gleiche Buttons | Speichern immer rechts unten |
| **Fehler vermeiden** | Design verhindert Fehler | Pflichtfelder markiert |
| **Offline-faehig** | Wichtige Funktionen ohne Internet | Monteur kann Fotos machen |
